#  FoodXpress – Swiggy/Zomato Clone

 A modern **Food Ordering Web App** inspired by Swiggy, Zomato, and Domino’s.  
Built using **HTML, CSS, and JavaScript**, featuring real-time cart updates, search filters, and a smooth checkout experience.

##  Features

 **Smart Search & Filters**
- Search food items instantly
- Category-based filtering (Pizza, Indian, Desserts, etc.)

 **Dynamic Cart System**
- Add/remove items
- Increase/decrease quantity
- Auto price calculation

 **Checkout Page**
- User details form
- Payment method selection
- Order summary

 **Local Storage**
- Cart persists after refresh

 **Responsive Design**
- Mobile-friendly UI
- Smooth animations
